EditorConfig 使用说明

2022-09-20

EditorConfig 使用说明

.EditorConfig 文件是为了保证跨 IDE 的代码一致性，以避免不同缩进方式带来的潜在安全问题

.EditorConfig 的优先级高于 IDE

查询表

通配方式

*

任何字符串，路径分隔符 ( / ) 除外

**

任何字符串

?

任何单个字符，路径分隔符 ( / ) 除外

[name]

匹配名称中的任何单个字符

[!name]

匹配名称中没有的任何单个字符

{s1,s2,s3}

匹配任何给定的字符串（以逗号分隔，可以嵌套）

{num1..num2}

num1 和之间的任何整数 num2，其中 num1 和 num2 可以是正数或负数

匹配到的不同的文件类型，可以定义不同的格式化属性。

特殊字符可以用反斜杠转义，这样它们就不会被解释为通配符模式。
